# REACT-Home-Page

### Introduction
- A clean boilerplate to start for projects built using nodejs, express and mongoDb

### [Live Demo](https://raga-react-home.netlify.app/)

### Installation

 - ```git clone https://github.com/RaghavJindal13/REACT-Home-Page.git ```
 - ```cd REACT Home Page```
 - ```npm install```
 - ```npm start```
